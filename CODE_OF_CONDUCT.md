# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 as its Code of Conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

Applies to all project spaces — repository, issues, pull requests, discussions, and any public or private communication where a participant is representing the project.

## Enforcement

Report concerns to the maintainers via a private channel (e.g., GitHub issue marked confidential, or email to the maintainer listed in `.claude-plugin/plugin.json`). Reports are reviewed promptly and handled with discretion.

## Attribution

Code of Conduct adapted from the Contributor Covenant, version 2.1, available at the URL above.
